Online Guitar & Music Instrument Store (MERN)

A full-featured MERN-stack e-commerce platform for purchasing guitars, guitar accessories, and other musical instruments.
Users can browse products, compare items, add to cart, and place secure orders.
Admins can manage products, categories, inventory, and orders.
